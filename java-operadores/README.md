## Operadores

- Atribuição `=`

- Aritméticos
	- adição `+` `Em caso de usado em Strings ele ira concatenar textos ou variaveis`
	- subtração `-`
	- multiplicação `*`
	- divisão `/`
	- modulo `%`

- Unário
	- Valor positivo `(+)`
	- Valor negativo `(-)`
	- Incremento de valor `(++)`
	- Decremento de valor `(--)`
	- Logico de Negação `!`

- Ternário
	- Comparador Verdadeiro `?`
	- Comparador Falso `:`

- Relacionais `Deve ser utilizado para variaveis nao Objeto`
	- Igualdade `==`
	- Diferença `!=`
	- Maior `>`
	- Menor `<`
	- Maior ou Igual `>=`
	- Menor ou Igual `<=`

- Lógicos
	- E `&&`
	- Ou `||`
